# Responsive Web Design with Bootstrap

A multi-page responsive website built using Bootstrap 4, demonstrating real-world layout patterns used in professional web applications.

## Pages

| Page | Description |
|---|---|
| `index.html` | Home / landing page |
| `about.html` | About section with responsive layout |
| `contact.html` | Contact form |
| `login.html` | User login form |
| `service.html` | Services listing |
| `employees.html` | Team / employee directory layout |

## Concepts Demonstrated

- **Responsive grid system** — 12-column Bootstrap grid adapting across mobile, tablet, and desktop
- **Bootstrap components** — navbar, cards, forms, buttons, modals
- **Custom CSS overrides** — `1bootstrap.css` extends Bootstrap defaults
- **Multi-page architecture** — shared navigation across all pages
